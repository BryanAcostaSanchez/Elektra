Resumen ejecutivo

Esta propuesta contempla el diseño y desarrollo de un sitio web promocional para el relanzamiento del capítulo "Mamá Sorda" dentro del universo de Historias de Elektra. El sitio será responsive y 100% mobile-first, diseñado para comunicar la historia del capítulo y amplificar su alcance digital.

El proyecto incluye:

  • Sitio web promocional responsive con diseño alineado a la identidad de Elektra.
  • Panel interno /admin para gestión de contenido, con filtros, búsqueda y exportación a Excel/CSV.
  • Integración con Google Tag Manager (GTM) y medición con GA4.
  • Reporte de rendimiento con métricas de GA4 al cierre del proyecto.
  • Hosting y dominio incluidos durante la vigencia del proyecto.

Arquitectura técnica

El sitio será construido con un stack moderno orientado a rendimiento y mantenibilidad:

  • Mobile-first & Responsive — Diseño 100% adaptable, optimizado primero para móviles.
  • Framework moderno — React/Next.js o similar, con componentes reutilizables y alto rendimiento.
  • Supabase (BaaS) — Backend para almacenamiento, autenticación y seguridad del panel admin.
  • GTM + GA4 — Google Tag Manager para orquestar eventos hacia GA4, Meta Pixel y otros.
  • Performance — Imágenes optimizadas, lazy loading, código minificado y fonts async.
